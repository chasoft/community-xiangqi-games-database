# 2017年07月04日重庆第五届“学府杯”象棋公开赛(229)

2017中国·重庆第五届“学府杯”象棋公开赛
竞赛规程
   
     一、指导单位：中国象棋协会
        二、主办单位：重庆市棋院
                  重庆市沙坪坝区教育委员会  
                  重庆市沙坪坝区体育局 
                  重庆市沙坪坝区陈家桥街道办事处
        三、承办单位：重庆大学城第三中学校
        四、协办单位: 沙坪坝区象棋协会
        五、竞赛时间、地点：比赛定于2017年7月4-7日在重庆大学城第三中学校（沙坪坝区陈家桥街道环城路）举行，比赛设成年组和少儿组分开比赛。首轮比赛自7月4日上午8:30开始。
请参赛棋手务必于7月3日18：00前现场签到，过时未签到一律不予编排。报名确认后无故不参赛者视作“非诚信棋手”，将被取消以后参赛资格。
  六、竞赛办法：采用《象棋竞赛规则（2011）》，实行积分编排制+淘汰制，成年组13轮，少儿组9轮，比赛单方用时35m+5s。
 七、报名资格、方式及费用：
1．成年组限报140人，重庆市内限报60名，市外80名，额满即止。资格要求：象棋特级大师及大师；全国业余高手或省级名手；重庆市级名手及重庆市属各区县冠军；港澳台、海外及女子棋手资格适当放宽。报名选手要求70岁以下且身体健康，品行优良，拒绝“非诚信棋手”参赛。选手报名时需交报名费100元/人，另交工作餐费100元/人（比赛期间午、晚餐及聚餐费用）。
2．少儿组（14岁以下）限报40人。原则上是重庆籍户口，在2003年1月1日之后出生的少年儿童，选手需交报名费100元/人，另交工作餐费100元/人（比赛期间午、晚餐及聚餐），陪同少儿参赛的教练或监护人交就餐费100元/人。
3．相关费用：所有参赛选手及监护人往返交通、住宿费和早餐费用自理（组委会为参赛选手统一提供比赛期间工作午餐、晚餐、比赛结束聚餐；全国冠军、港澳台及海外棋手补贴500元交通费），安全责任自负；组委会为所有参赛棋手提供比赛服装，赠送精美纪念品。少儿选手监护人必须提前购买相关责任保险，否则责任自担。
4．报名方式：报名必须注明姓名、性别、组别、省别、个人等级或历史最好成绩，以短信方式发到王钊远（逍遥君子）手机13883873280上，或者通过关注指定微信公众号“SPORT重庆”，然后注册登录查找相应赛事后根据提示提交上述信息。报名截止于2017年6月28日18：00。所有报名人员须经组委会审核同意后发布到广东象棋网上的人员，方可届时现场报到交费参赛。
八、奖励设置：总奖金112000元。
1．少儿组：奖励前12名选手，发奖金和证书，冠军奖金600元，亚军500元，季军400元，第4名300元，第5名200元，第6-12名各奖100元。另：2009年1月1日后出生的儿童单独排名，奖励前6名，第1名300元，第2名200元，第3-6名各100元。所有参赛选手均颁发参赛证书，少儿组奖金合计3600元。
2．成年组：录取前48名选手并颁发奖金和证书，其中前三名单独颁发奖杯。奖金分配如下：冠军20000元，亚军10000元，季军6000元，第4名5000元，第5名4000元，第6名3500元，第7名3000元，第8名2500元，第9名2200元，第10名2000元，第11-16名顺次为1800元，1800元，1600元，1600元，1400元，1400元，第17-20名各1200元，第21-24名各1100元，第25-32名各1000元，第33-37名各800元，第38-42名各600元，第43-48名各400元。成年组奖金合计94400元。
3．本地奖：奖励成年组重庆籍（以身份证为准）选手前20名，颁发证书及奖金。第1名2000元，第2名1500元，第3名1200元，第4名1000元，第5名800元，第6名600元，第7名500元，第8名各400元，9-12名各300元，13-20名各200元。重庆本地选手奖金合计10800元。
4．女子奖：取成年女子前六名颁发证书及奖金。按名次依次为1000元、800元、500元、400元、300元和200元。女子组奖金合计3200元。
 九、组委会聘请胡智慧担任裁判长，黄政为编排长。网络报道：逍遥君子。
 十、未尽事宜按组委会赛前另行制定的《补充规定》执行。
